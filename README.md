# Football Quiz
Football Quiz is a mobile application that allows users to test their knowledge about football. The application contains questions divided into categories: World Cup, Champions League, Polish Football, and Curiosities. Users can play with or without background music and sound effects. After answering all the questions, the user can share their results through Messenger.

The app was created in Android Studio using Kotlin programming language. 

# Categories
The quiz is divided into the following categories:
- World Cup,
- Champions League,
- Polish Football,
- Curiosities.

# Features
The application provides the following features:
- CSV reading
- Audio management (background music and sound effects)
- Sharing the quiz result via Messenger
